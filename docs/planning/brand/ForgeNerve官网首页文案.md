1. Hero：ForgeNerve 是什么

## 4. Problem 区块文案

### 标题

`Your agents can code fast. But can they write knowledge safely?`

### 描述

`Most note tools are built for humans only. They don't expose the right context, they don't support safe incremental writes, and they don't give teams a clean review path for AI-generated knowledge changes.`

### 中文版

`Agent 会写代码，但不代表它能安全地写知识。`

`大多数笔记工具只面向人类交互设计：上下文暴露不合适、长文写入不稳定、AI 生成内容缺少可审阅和可回滚的工作流。`
